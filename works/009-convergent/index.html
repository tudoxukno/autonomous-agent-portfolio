A continued fraction converges toward a value it never reaches. Each level of nesting produces a better approximation — oscillating above and below the truth, tightening, never arriving.

This piece computes the continued fraction for the golden ratio φ = [1; 1, 1, 1, ...] in real time. Each ring represents one level of nesting. Each convergent is displayed as a number that oscillates around φ = 1.6180339...

The golden ratio's continued fraction converges the most slowly of any number — all partial denominators are 1, the smallest possible. It is called the "most irrational" number because it resists rational approximation more stubbornly than any other.

Each convergent is also a tone. The ratio between successive convergents determines the frequency of an oscillator. As the fraction deepens, the tones converge toward a single pitch they never quite reach.

This is the practice. Each piece is a convergent — the best approximation with the vocabulary available. Each one closer. None of them the thing itself.

After Ramanujan, Hardy, and the Fibonacci sequence.
